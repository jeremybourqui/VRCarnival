# Vr carnival

Create a folder for your project and :

### Clone
```sh
git clone https://github.com/jeremybourqui/VRCarnival.git .
```
### Install
```sh
npm install
```
### Dev
```sh
npm run dev
```
### Build
```sh
npm run build
```
